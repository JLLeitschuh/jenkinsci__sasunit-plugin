Pfad zum Wurzelverzeichnis der SASUnit Version, die verfügbar sein soll. Der Pfad wird validiert und geprüft, ob dieser auf dem Master existiert.
Dieser Pfad wird als Default-Wert an alle Knoten weitergegeben. Sollte eine SASUnit Installation auf einem Slave-Knoten unter einem anderen Pfad erreichbar sein, kann der Default-Wert in der Konfiguration des Slave-Knoten. überschrieben werden.
